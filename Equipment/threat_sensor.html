Threat Sensor

It is also available only in multiplayer mode and it is essential to utilize this tool to have an idea on where your enemies are located. The function of this tool is to highlight enemies in a certain area. You may locate and see through enemies (which will be a red form) using the sensor dart that will last around 8 seconds.

~Maximum number of uses: 3~
